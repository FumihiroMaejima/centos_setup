# CentOS Setup Environmental Version7

CentOS7の環境構築方法をまとめる

## OSのバージョンのチェック

```shell-session
# cat /etc/system-release
```

## rootパスワードの変更

```shell-session
# passwd
```

## ユーザーの作成

```shell-session
# useradd test_user
```

## 作成したユーザーのパスワードの追加

```shell-session
# passwd test_user
```
